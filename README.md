# CRM для риэлторов и агентств недвижимости

Веб-приложение для управления клиентами, объектами и сделками. React + TypeScript + Vite + Supabase.

## Возможности

- Клиенты, объекты, сделки
- CRUD для всех сущностей
- Авторизация и роли
- Дашборд и отчёты
- Тёмная/светлая тема

## Разработка

```bash
cp .env.example .env   # Заполните Supabase ключи
npm install
npm run dev
```
